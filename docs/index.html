Convertidor TXT-XML

Software libre para conversión de listas de picos MALDI-TOF del tipo generado por Vitek MS / Saramis RUO a XML compatible con MicrobeNet.

Descripción / Description

Software libre para la conversión de listas de picos MALDI-TOF del tipo generado por Vitek MS / Saramis RUO, desde archivos .txt a formato XML compatible con la base de datos pública Microbenet (microbenet.cdc.gov). Software liviano, de ejecución local, con soporte drag & drop para procesamiento simple y rápido. Registrado ante la Dirección Nacional de Derechos de Autor (DNDA), Ley 11723 de Propiedad Intelectual, República Argentina (2025). Open source software for converting MALDI-TOF peak lists of the type generated by Vitek MS / Saramis RUO, from .txt files to XML format compatible with the public MicrobeNet database (microbenet.cdc.gov). Lightweight, locally executed software with drag-and-drop support for fast and simple processing. Registered with the Dirección Nacional de Derechos de Autor (Ley 11723), República Argentina (2025).
