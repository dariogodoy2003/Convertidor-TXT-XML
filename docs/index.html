Convertidor TXT-XML

Software libre para la conversión de listas de picos MALDI-TOF del tipo generado por Vitek MS / Saramis RUO, desde archivos .txt a formato XML compatible con la base de datos pública Microbenet (microbenet.cdc.gov). Software liviano, de ejecución local, con soporte drag & drop para procesamiento simple y rápido.
