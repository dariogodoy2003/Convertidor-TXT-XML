# Convertidor-TXT-XML
Software libre para la conversión de listas de picos de espectros de masa MALDI-TOF del tipo Vitek MS, en formato .txt, a formato XML compatible con la base de datos pública Microbenet (microbenet.cdc.gov).

# TXT-XML-Converter
Free software for converting MALDI-TOF mass spectrum peak lists of the Vitek MS type from .txt format to XML format compatible with the public MicrobeNet database (microbenet.cdc.gov).
